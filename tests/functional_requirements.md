# Functional Requirements

These functional requirements describe what the Air Pollution & Health Impact Dashboard must do for users.

## FR1 – Overview information
Users shall be able to view an overview section that explains the purpose of the dashboard and how the air pollution health indicators should be interpreted.

## FR2 – Key metrics
Users shall be able to view key summary metrics for a selected indicator and a selected year, including an **All years** option.

## FR3 – Top regions chart
Users shall be able to view the top 10 most impacted regions for a selected year and indicator.

## FR4 – Impact map
Users shall be able to view country-level impact on an interactive Europe map for a selected year and indicator.

## FR5 – Region comparison
Users shall be able to compare the impact values of selected regions for a chosen year and indicator.

## FR6 – Trend analysis
Users shall be able to view trends over time for a selected indicator, either for all regions combined or for one selected region.

## FR7 – Filtered data table
Users shall be able to view a filtered data table based on the current key metrics selection.
